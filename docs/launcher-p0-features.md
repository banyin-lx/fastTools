# 快捷启动软件 P0 功能清单

## 1. 文档说明

本文档基于 [launcher-product-plan.md](D:/code/fastTools/docs/launcher-product-plan.md) 中的建议功能优先级整理，只保留 `P0` 范围，用于：

- 明确首版必须交付的功能
- 作为开发排期与任务拆分依据
- 作为测试验收的基础清单

`P0` 目标不是“功能最多”，而是优先做出一版 **可稳定使用、响应快、UI 完整、可继续扩展** 的桌面启动器。

---

## 2. P0 功能范围总览

P0 必须包含以下模块：

- 全局呼出
- 搜索框与结果列表
- 应用启动
- 文件检索
- 网页搜索
- 系统命令
- 设置页
- 基础主题能力
- 本地插件系统

---

## 3. P0 详细功能

## 3.1 全局呼出

### 功能说明

用户可通过全局快捷键在任意界面呼出启动器主窗口，并在失焦后自动隐藏。

### 功能点

- 支持默认全局快捷键，例如 `Alt + Space`
- 支持在设置页修改全局快捷键
- 支持程序单实例运行，重复启动时聚焦到已运行实例
- 主窗口支持居中显示
- 主窗口失焦时自动隐藏
- 支持按 `Esc` 关闭窗口

### 验收标准

- 在桌面、浏览器、编辑器等常见应用中都可正常呼出
- 快捷键冲突时能提示用户重新设置
- 连续多次呼出 / 关闭不会出现窗口卡死或焦点异常

---

## 3.2 搜索框与结果列表

### 功能说明

主窗口以搜索框为核心，用户输入关键字后实时展示结果，并可通过键盘快速选择和执行。

### 功能点

- 顶部搜索输入框自动聚焦
- 输入关键字后实时刷新结果
- 结果列表支持键盘上下移动选择
- 支持 `Enter` 执行当前选中项
- 支持结果分组展示：
  - 应用
  - 文件
  - 网页
  - 系统命令
  - 插件结果
- 支持空状态提示
- 支持最近使用记录参与排序
- 支持模糊匹配
- 支持拼音首字母匹配
- 支持别名匹配

### 验收标准

- 常见搜索请求应在可接受时间内返回结果
- 键盘操作流程完整，不依赖鼠标
- 搜索无结果时界面状态清晰，不出现空白异常

---

## 3.3 应用启动

### 功能说明

用户可通过关键字快速搜索和启动本机应用。

### 功能点

- 索引系统开始菜单应用
- 索引桌面快捷方式
- 支持配置额外应用目录
- 支持识别并启动以下目标：
  - `.exe`
  - `.lnk`
  - 常见 UWP 应用
- 支持“以管理员身份运行”
- 支持应用别名配置
- 支持展示应用图标、名称、路径

### 验收标准

- 常用应用可被正确索引并搜索到
- 启动路径正确，不出现误启动
- 管理员方式运行可正常触发 UAC

---

## 3.4 文件检索

### 功能说明

用户可在指定目录范围内搜索文件与文件夹，并执行基础文件动作。

### 功能点

- 支持配置索引目录
- 支持按文件名检索文件 / 文件夹
- 支持常见文件类型结果展示
- 支持基础动作：
  - 打开文件
  - 打开文件夹
  - 打开所在目录
  - 复制完整路径
  - 使用默认程序打开
- 支持显示文件图标、名称、所在路径

### 验收标准

- 只在用户配置目录内建立索引
- 文件结果点击或回车后能执行正确动作
- 大目录下搜索体验可接受，不出现明显卡顿

---

## 3.5 网页搜索

### 功能说明

用户可输入网址直接打开，或将普通关键字提交到默认搜索引擎。

### 功能点

- 支持识别 URL 并直接用默认浏览器打开
- 支持将非 URL 文本作为搜索词打开
- 支持配置默认搜索引擎
- 支持自定义搜索源，例如：
  - 百度
  - Google
  - Bing
  - GitHub
  - Bilibili
- 支持通过前缀关键字触发特定搜索源

### 验收标准

- 网址识别准确，不误判普通文本
- 搜索请求可稳定拉起默认浏览器
- 搜索源配置后可立即生效

---

## 3.6 系统命令

### 功能说明

用户可快速执行常见系统操作与本地命令。

### 功能点

- 提供内置系统动作：
  - 锁屏
  - 重启
  - 关机
  - 睡眠
  - 打开任务管理器
  - 打开控制面板
  - 打开 Windows 设置
  - 清空回收站
- 支持用户配置自定义命令
- 支持用户配置脚本入口
- 支持执行前的风险确认，用于高风险动作

### 验收标准

- 内置系统命令执行结果正确
- 高风险操作在触发前有明确确认
- 自定义命令配置后可被搜索和执行

---

## 3.7 设置页

### 功能说明

提供一个统一设置中心，用于管理启动器核心行为。

### 功能点

- 全局快捷键设置
- 索引目录设置
- 搜索排序相关设置
- 主题设置
- 插件开关
- 自定义命令管理
- 基础关于页或版本信息展示

### 验收标准

- 所有 P0 核心设置项都能在设置页完成配置
- 设置修改后可正确持久化
- 重启程序后配置仍然生效

---

## 3.8 基础主题能力

### 功能说明

首版需要提供完整且统一的现代化视觉风格，并支持基础主题切换。

### 功能点

- 支持浅色主题
- 支持深色主题
- 主窗口、结果项、设置页视觉统一
- 支持圆角、阴影、层次化布局
- 支持基础窗口过渡动画
- 支持选中态、悬浮态、禁用态视觉规范

### 验收标准

- 浅色 / 深色主题切换后界面元素显示正常
- 不同页面和组件视觉风格一致
- 动画存在但不拖慢交互响应

---

## 3.9 本地插件系统

### 功能说明

P0 阶段只做本地插件系统基础版，用于保证产品后续可扩展，同时将部分搜索能力标准化。

### 功能点

- 定义统一插件接口
- 支持本地插件发现与加载
- 插件可返回搜索结果项
- 插件可声明名称、版本、描述
- 插件支持启用 / 禁用
- 网页搜索、部分系统命令能力可按插件方式接入
- 系统命令/网页搜索功能 两个模块为内置插件

### 验收标准

- 主程序启动时可正确加载本地插件
- 插件异常不会导致主程序崩溃
- 禁用插件后，相关结果不再显示

---

## 4. P0 不包含的内容

以下内容不进入首版上线范围：

- 在线插件市场
- 剪贴板历史
- 快捷短语
- 排序学习与复杂个性化推荐
- 工作流编排
- AI 能力
- 云同步

---

## 5. P0 建议的开发拆分

可按以下顺序推进：

1. 程序壳层与全局呼出
2. 主窗口、搜索框、结果列表 UI
3. 搜索核心与排序机制
4. 应用索引与启动
5. 文件索引与文件动作
6. 网页搜索与系统命令
7. 设置页与配置持久化
8. 本地插件系统
9. 联调、性能优化、异常处理

---

## 6. P0 交付定义

当以下条件同时满足时，可认定 `P0` 完成：

- 用户可通过全局快捷键稳定呼出启动器
- 用户可使用键盘完成搜索、选择、执行
- 应用、文件、网页、系统命令四类核心结果可正常使用
- 设置页可完成核心配置管理
- 主界面具备统一现代化视觉风格
- 本地插件系统可正常加载基础插件
- 主要流程稳定，无明显阻塞级 Bug
